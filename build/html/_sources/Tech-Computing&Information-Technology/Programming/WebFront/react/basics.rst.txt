What is React
==============

A JavaScript library for building user interfaces.

Basically, React is a Single Page Application (SPA) framework, which means that everything run through one single page (i.e. index.html).

- makes front-end JavaScript much easier
- uses self contained, independent components with their own state
- much more interactive UIs
- virtual DOM
- JSX - easily incorporate JS in markup

------------

- JavaScript
