
Python Web Framworks
======================

Python 中 web framework 的大致结构:

1. 收发 socket 消息, 按照 HTTP 协议解析
2. 根据路径执行对应的函数
3. 返回模板文件, 字符串替换

请求流程:

1. 服务端启动, 等待客户端连接
2. 浏览器客户端输入 URL 地址, 与服务端建立连接, 并发送请求
3. 服务端接受请求并解析, 根据请求内容找到对应的函数
4. 执行函数, 得到需要返回的内容
5. 按照 HTTP 协议封装内容, 并返回
6. 客户端接受相应消息, 按照内容渲染
7. 断开连接
